# Minecraft Optimization Mods

## ※整備中

Minecraftを快適に遊ぶためのいわゆる軽量化Modリスト

### 但し書き
- あくまで全ての軽量化Modのリストを作成したいという考えに基づいて作成しているため、このリストに載っている全てのModを導入することは推奨しません
- 便利系やバグ修正など軽量化を目的としていないModはあえて掲載していません
